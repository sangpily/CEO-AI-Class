A. ChatGPT 트랙 (3시간 / 180분)
대표 업무를 대신하는 "AI직원(GPTs)"을 실제로 만들어, 회사 운영에 바로 투입
10분

A0) 준비 & 목표 정의

  • 로그인/환경 점검(ChatGPT)
  • "오늘 AI로 해결할 대표 업무 1개" 한 줄 정의
15분

A1) CEO 맞춤 세팅(맞춤설정)

대표 말투/보고 포맷 고정

  • 맞춤설정(Custom Instructions)로 "CEO 비서/기획실장"화
  • 출력 규격 고정: 표/요약/체크리스트/액션아이템 중심
25분

A2) 프로젝트(Project) 기능 실습

"내 회사 AI 운영실" 구축

프로젝트에 고정할 4가지 입력:
  1. 회사/제품/고객/가격/강점
  2. 산출물 포맷(1페이지 보고서/회의록/제안서)
  3. KPI 3~5개
  4. 금지/주의(민감정보, 확정표현 금지 등)
20분

A3) 회의·상담 기록 자동화

"요약→결정→할일" 프로세스

  • 실제 메모/카톡/메일 일부(민감정보 제거)로
  • 1페이지 실행 회의록 생성
  • 담당/기한/후속 메시지 초안 작성
30분

A4) ADA(Advanced Data Analysis)

경영 숫자 리포트 자동화

  • 숫자 5개 또는 CSV/엑셀(가능자)로
  • 표/차트 + 비용 구조 + 개선안 도출
5분

A4-2) Task 기능 NEW

반복 업무 자동 실행 구조

  • Task의 목적: 매주/매월 반복되는 대표 업무를 자동 실행
  • 실습(데모): 매월 1일 "월간 리포트 생성" / 매주 월요일 "주간 우선순위 5개 + 실행계획"
  • 결과: 프로젝트와 연결해 반복 운영 루틴 만들기
75분

A5) 핵심: "내 회사 AI직원(GPTs) 제작" 실습 CORE

GPTs 1개 완성 + 현장 튜닝까지 진행

1. AI직원 직무 선택(택1)
  • 재무담당 / 영업팀장 / 마케팅팀장 / 기획실장
2. GPTs 설계도 작성
  • 입력(고정): 회사소개/고객/가격/강점/경쟁사/사례
  • 출력(고정): "1페이지 보고서/표/액션 5개"
  • 규칙: "근거 → 결론 → 실행" 구조 강제
3. GPTs 빌드 + 테스트 시나리오 5개
  • 리포트, 비용절감 플랜, 우선순위 선정, 원인 가설, 지시문 생성
4. 현장 수정/튜닝
  • 요약 규칙 강화, KPI/제약조건 추가, 액션 포맷 강제
B. Gemini 트랙 (1시간 / 60분)
CEO가 바로 활용하는 "콘텐츠 생산 + 자료 기반 보고/방송 자동화"

B0) Gemini 장점 3가지 NEW

대표 체감 기준

  • 멀티모달: 이미지/문서/표 등 다양한 입력을 함께 이해하고 산출물로 연결
  • 아이데이션/변주 속도: 타깃별 메시지·카피·구성안을 빠르게 여러 버전으로 생성
  • 콘텐츠 패키징: 스크립트/슬라이드/요약본 등 여러 형식으로 재가공(재활용) 용이
45분

B1) NotebookLM RAG 실습

회사 자료로 "보고서→콘텐츠" 만들기

NotebookLM을 RAG(사내자료 기반 검색·근거 생성) 용도로 사용

입력 자료 예시:

회사소개서/브로셔/제안서/정책/성과자료/인터뷰 텍스트 등

실습 흐름:
  1. 자료 업로드 및 핵심 근거(근거 기반 요약) 추출
  2. 그 근거를 바탕으로 PPT 보고서 슬라이드 10~12장 구성안 생성
  3. 동일 내용을 동영상(홍보/설명) 스크립트로 변환(60~90초)
  4. 자료를 팟캐스트 원고(10~15분)로 재구성(오프닝-본론-클로징-CTA)
산출물: 회사 자료 기반
  • PPT 보고서 원고(슬라이드별 텍스트)
  • 동영상 스크립트(짧은 버전)
  • 팟캐스트 원고 1편
10분

B2) Gemini로 최종 다듬기

문장/톤/타깃별 버전 조정

  • PPT/영상/팟캐스트의 문장 톤을 대표 스타일로 정리
  • 타깃 2종(예: 가격민감/품질중시)으로 메시지 변주

용어 정리

핵심 개념 이해하기

프로젝트(Project)
회사별 대화·자료·규칙·산출물을 모아 "업무 공간"처럼 쓰는 기능입니다. 특정 업무나 프로젝트에 필요한 모든 컨텍스트와 설정을 한 곳에 저장하여 일관된 결과물을 생성할 수 있습니다.
ADA (Advanced Data Analysis)
엑셀/CSV 데이터를 분석해 표·차트를 만드는 기능입니다. 복잡한 데이터 분석을 자동화하여 경영 의사결정에 필요한 인사이트를 빠르게 도출할 수 있습니다.
Task
매주/매월 반복되는 업무를 자동 실행하도록 설정하는 기능입니다. 정기적인 리포트 생성이나 체크리스트 작성 등 루틴 업무를 자동화할 수 있습니다.
GPTs (맞춤형 GPT)
특정 업무 역할(재무담당/영업팀장 등)을 수행하도록 규칙과 출력 형식을 고정한 전용 챗봇입니다. 회사의 업무 프로세스에 맞게 커스터마이징된 AI 직원을 만들 수 있습니다.
RAG (Retrieval-Augmented Generation)
사내 문서/자료를 기반으로 필요한 정보를 찾아(검색) 근거 있게 답변·산출물을 생성하는 방식입니다. 회사의 축적된 지식을 AI가 활용할 수 있게 합니다.
NotebookLM
문서 기반으로 근거를 뽑아 요약/질문답변/콘텐츠 재가공을 돕는 도구입니다. 기존 자료를 다양한 형태의 새로운 콘텐츠로 변환할 수 있습니다.
멀티모달 (Multimodal)
텍스트뿐 아니라 이미지·문서·표 등 다양한 입력을 함께 처리하는 방식입니다. 여러 형태의 정보를 통합적으로 분석하고 활용할 수 있습니다.
