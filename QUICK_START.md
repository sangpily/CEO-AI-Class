## 📝 3단계로 끝!

### ✅ 1단계: GitHub에 Repository 만들기 (2분)

1. https://github.com 로그인
2. 우측 상단 `+` → `New repository`
3. **Repository name**: `ceo-ai-class` ⚠️ 정확히 입력!
4. **Public** 선택
5. **Create repository** 클릭

---

### ✅ 2단계: 파일 업로드 (2분)

**방법 A: 웹에서 업로드 (추천 - 가장 쉬움)**

1. 다운로드한 ZIP 파일 압축 해제
2. `ceo-ai-class` 폴더 안의 파일 확인:
   - index.html
   - README.md
   - DEPLOY_GUIDE.md
3. GitHub repository 페이지에서:
   - `uploading an existing file` 링크 클릭
   - 위 3개 파일 드래그 앤 드롭
   - 하단 `Commit changes` 버튼 클릭

**방법 B: Git 명령어 사용 (기술적)**

```bash
cd ceo-ai-class
git remote add origin https://github.com/sangpily/ceo-ai-class.git
git push -u origin main
```

---

### ✅ 3단계: GitHub Pages 활성화 (1분)

1. Repository 페이지에서 **Settings** 탭
2. 좌측 **Pages** 메뉴
3. **Branch**: `main` 선택 → **Save**
4. **완료!** 1~2분 후 사이트 주소 확인
